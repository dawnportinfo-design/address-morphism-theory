# Frontmatter

**Title:** Address Morphism Theory: A Formal Framework for Address Reference,
Registration, Resolution, And Safe Identifier Issuance

**Short title:** Address Morphism Theory

**Repository role:** This paper series defines the theory layer. AGID can use
the theory to implement resolvers, postal-zone generation, private QR, SDKs,
and operational workflows, but this repository does not claim production
coverage by itself.

**Reader promise:** Every chapter should state what problem it solves, what it
does not claim, and which executable model or appendix supports it.
