# 16. Natural Geography, Cultural Geography, And Vertical Reference

Not every addressable entity is a building. Seas, islands, ports, mountains,
deserts, rivers, cultural sites, floors, entrances, lockers, and drone handoff
zones can all require address-like references.

AMT supports these through typed referents and bounded regions, while avoiding
claims that every named feature is globally verified.

Model hook: `formal/definitions.ts`, `formal/verification-map.ts`.
