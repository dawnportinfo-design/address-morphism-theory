# Validation and Open Claims

AMT separates formal claims from empirical coverage claims.

## Validated Locally

The repository contains executable models for:

- ambiguous candidate handling
- unresolved candidate omission
- evidence gate rejection
- no-postcode AGID-first handling
- purpose-relative resolution
- lineage functionality
- predicate anonymity-set checks

## Not Yet Globally Validated

The project must not claim:

- complete worldwide candidate generation
- complete multilingual recall
- complete natural-feature recognition
- strict GIS warning-free global validation
- superiority over commercial validation APIs
- audited production zero-knowledge security

These are research and verification targets, not completed claims.
