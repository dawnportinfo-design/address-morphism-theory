# Appendix C. Counterexamples

## Same String, Different Referents

Two buildings can share a visible local name in different districts.

## Same Coordinate, Different Operational Referents

A building centroid may not identify the correct entrance, locker, room, or
delivery bay.

## Postal Code Success, Delivery Failure

A postal code can be valid while the recipient's operational handoff point
remains unresolved.

## Translation Without Reference Preservation

Literal translation can turn a place name into an incorrect semantic phrase.

## Privacy-Preserving Proof Without Candidate Completeness

A cryptographic proof can prove membership in a committed set while the set
itself may be incomplete.
