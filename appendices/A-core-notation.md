# Appendix A. Core Notation

- `E`: surface address expressions.
- `N`: normalized expressions.
- `C`: candidate sets.
- `S`: source and evidence states.
- `R`: referent states.
- `P`: persistent identifiers.
- `V`: purpose-specific views.
- `ctx`: context, including jurisdiction, language, institution, time, and use.
- `f_ctx`: a context-indexed partial morphism.

Core chain:

```text
E -> N -> C -> S -> R -> P -> V
```
